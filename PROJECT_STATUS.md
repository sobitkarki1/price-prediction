## Architecture Details

### Model Configuration
```python
Input 1: Sequence (30 days, 14 features)
Input 2: Stock ID (integer 0-286)

LSTM(128) ‚Üí Dropout(0.3) ‚Üí
LSTM(64) ‚Üí Dropout(0.3) ‚Üí
LSTM(32) ‚Üí Dropout(0.3) ‚Üí

Stock Embedding(287, 16) ‚Üí

Concatenate(LSTM output, Stock embedding) ‚Üí
Dense(32, relu) ‚Üí Dropout(0.3) ‚Üí
Dense(16, relu) ‚Üí
Output(1) - Predicted price (5 days ahead)
```

### Hyperparameters
- **Lookback**: 30 days
- **Forecast horizon**: 5 days ahead
- **Batch size**: 64
- **Max epochs**: 150
- **Early stopping**: Patience 20
- **Optimizer**: Adam
- **Loss**: MSE
- **Embedding dim**: 16 (increased from 8)
- **Dropout**: 0.3 (increased from 0.2)

### Features (14)
1. Open, High, Low, Close, Volume
2. return, volume_change
3. sma_5, sma_10, sma_20
4. std_5, std_10, std_20
5. rsi
